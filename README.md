# StudyBuddy

StudyBuddy is a simple homework task management web application created for a Computer Science final project.

## Features

- Add homework tasks
- Delete tasks
- Mark tasks as completed
- Save tasks using LocalStorage
- Responsive and simple design

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage
- GitHub Pages

## How to Run the Project

1. Download or clone the repository.
2. Open the folder in Visual Studio Code.
3. Open index.html in your browser.

## GitHub Pages Hosting

The project can be hosted using GitHub Pages.

1. Upload the files to a GitHub repository.
2. Go to Settings.
3. Select Pages.
4. Choose the main branch.
5. Save the settings.
6. GitHub will generate a public website link.
